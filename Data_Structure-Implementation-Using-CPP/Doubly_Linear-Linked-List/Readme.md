# Doubly_Linear_Linked_List (C++)

This repository contains a **Doubly Linear Linked List** implementation in **C++ language**. It is designed for learning and practicing **Data Structures** concepts such as pointers, dynamic memory allocation, and linked list traversal using object-oriented programming.

A doubly linear linked list is a collection of nodes where:

* Each node stores **data**
* Each node contains a pointer to the **next node**
* Each node contains a pointer to the **previous node**
* The last node points to **NULL**

---

##  Data Structure Used

```cpp
struct node
{
    int data;
    node *next;
    node *prev;
};
```
## Operations Implemented
Insert node at first position

Insert node at last position

Insert node at given position

Delete node from first position

Delete node from last position

Delete node from given position

Display all elements (forward & backward)

Count number of nodes
