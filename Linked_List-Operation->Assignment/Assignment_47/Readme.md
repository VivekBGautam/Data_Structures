# Assignment_47

Here contains multiple **C programs** that perform various operations on a **Singly Linear Linked List**.  
These programs focus on **reverse traversal, digit operations, and prime number operations**, helping to understand advanced linked list manipulations.

---

## Programs Overview

- **program47_1.c** – Prints the linked list **in reverse order**.  
  **Function Prototype:** `void DisplayReverse(PNODE Head);`

- **program47_2.c** – Counts the **number of digits** in each node.  
  **Example:** `123 -> 3 digits`  
  **Function Prototype:** `void CountDigits(PNODE Head);`

- **program47_3.c** – Calculates and displays the **sum of digits** of each node.  
  **Function Prototype:** `void SumDigits(PNODE Head);`

- **program47_4.c** – Displays all **prime numbers** present in the linked list.  
  **Function Prototype:** `void DisplayPrime(PNODE Head);`

- **program47_5.c** – Counts the **total number of prime numbers** in the linked list.  
  **Function Prototype:** `int CountPrime(PNODE Head);`
