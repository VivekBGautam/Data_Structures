# Assignment_48

Here contains multiple **C programs** that perform various operations on a **Singly Linear Linked List**.  
These programs focus on **element replacement, modification, and position-based display**, helping to understand advanced linked list manipulation techniques.

---

## Programs Overview

- **program48_1.c** – Replaces all **even numbers** in the linked list with `0`.  
  **Function Prototype:** `void ReplaceEven(PPNODE Head);`

- **program48_2.c** – Replaces all **odd numbers** in the linked list with `1`.  
  **Function Prototype:** `void ReplaceOdd(PPNODE Head);`

- **program48_3.c** – Multiplies **each element** of the linked list by `2`.  
  **Function Prototype:** `void MultiplyByTwo(PPNODE Head);`

- **program48_4.c** – Displays elements present at **even positions** in the linked list.  
  **Function Prototype:** `void DisplayEvenPosition(PNODE Head);`

- **program48_5.c** – Displays elements present at **odd positions** in the linked list.  
  **Function Prototype:** `void DisplayOddPosition(PNODE Head);`
